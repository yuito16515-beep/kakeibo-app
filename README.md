# 家計簿アプリ

## 概要
収入と支出を記録し、合計金額や収支バランスを確認できる家計簿アプリです。  
月別フィルター機能や編集機能を実装し、実用性を意識して制作しました。

## 使用技術
- HTML
- CSS
- JavaScript
- localStorage

## 実装機能
- 収入・支出の追加（Create）
- データの編集（Update）
- データの削除（Delete）
- 月別フィルター機能
- 合計収入・支出・収支の自動計算
- データの保存（localStorage）

## 工夫した点
- date型を利用した月フィルターの実装
- reduce を用いた合計金額の計算
- 編集時にフォームへ値を戻す処理
- マイナス収支時の色変更で視認性向上
