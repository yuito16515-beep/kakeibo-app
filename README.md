# 家計簿アプリ

## ■ 概要
収入・支出を記録し、合計金額と収支を自動計算できるシンプルな家計簿アプリです。  
ブラウザの Web Storage API（localStorage）を使用してデータを保存するため、ページを再読み込みしてもデータが保持されます。

JavaScriptによるCRUD機能（作成・表示・更新・削除）を実装したフロントエンドアプリケーションです。

---

## ■ 公開URL
https://（あなたのユーザー名）.github.io/kakeibo-app/

## ■ 主な機能

- 収入・支出の登録（Create）
- 一覧表示（Read）
- 編集機能（Update）
- 削除機能（Delete）
- 合計収入・支出・収支の自動計算
- データの永続化（localStorage）

---

## ■ 使用技術

- HTML5
- CSS3
- JavaScript（ES6）
- Web Storage API（localStorage）
- Git / GitHub Pages

---

## ■ 設計意図

- 配列によるデータ管理を行い、状態変更後は再描画（render関数）でUIを更新する構成にしました。
- localStorageにJSON形式で保存することで、簡易的なデータ永続化を実現しています。
- CRUDの基本操作を明確に分けることで、実務でのデータ処理を意識した構成にしています。

---

## ■ 工夫した点

- 編集時にボタン表示を「追加」から「更新」に切り替えることで、状態を明確にしました。
- 合計金額・収支を自動計算し、データ変更時に即時反映されるよう設計しました。
- シンプルなUIにしつつ、入力と表示の役割を分離しています。
